# **App Name**: JobTracker Pro

## Core Features:

- Dashboard: Displays key metrics like total applications, active interviews, geographic breakdown, and success rate.
- Application List: A sortable and filterable list displaying key details of each job application, including company, role, date applied, location, and current status.
- Add Application Modal: A modal form to create new job applications with fields for company, role, job description link, application date, and location.
- Smart Location Logic: The application analyzes the location input to identify "Remote" roles or specific country locations.
- JD Scanner Tool: Uses AI to scan the provided job description link, attempts to extract the location from the url and prepopulates the location field in the form.
- Custom Progress Pipeline: Tracks each application through a defined 9-stage pipeline (Applied, Recruiter Call, etc.)
- Automatic Archiving: Automatically archives applications in 'No Offer' stage, moving them to the 'Archived/Rejected' tab while still counting them in total applications.

## Style Guidelines:

- Primary color: Stormy Teal (#15616D), for a professional and sophisticated feel.
- Background color: Ink Black (#001524) for a minimalist dark mode aesthetic.
- Accent color: Papaya Whip (#FFECD1) to highlight interactive elements and calls to action.
- Font color: White (#FFFFFF) for high contrast and readability on the darker backgrounds and use Ink Black whenever there is text on the Accent color as background.
- Body and headline font: 'Source Sans 3', a sans-serif with a modern and neutral look.
- Lucide icons will be used for a clean and consistent look throughout the application.
- A clean, minimalist layout optimized for both desktop and mobile views.
- Subtle transitions and animations will be incorporated to enhance the user experience.
